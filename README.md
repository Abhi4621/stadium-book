A modern, high-energy **Sports Stadium Booking Platform** landing page inspired by the premium aesthetic of top-tier sports organizations like the BCCI and UEFA. This project features a "Midnight Stadium" theme with immersive animations and a production-ready booking portal.

---

## üåü Key Features

* **Sticky Glass Navbar**: Features dynamic backdrop blur and shadow effects that trigger upon scrolling.
* **Hero Section**: High-resolution stadium backdrop utilizing parallax effects and Framer Motion text-reveal animations.
* **Animated Stats Section**: Counter-style reveal for platform metrics (500+ Bookings, 50+ Stadiums) to build user trust.
* **Pro Booking Portal**: Interactive glassmorphism form with real-time state management, loading indicators, and success feedback.
* **Dynamic Atmosphere**: Custom CSS-based "Starry Night" background animation with twinkling effects.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop views using a mobile-first approach.
